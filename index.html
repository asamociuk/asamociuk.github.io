Intelligent BLDC servo motor controller with high-level commands

The aim of the thesis was to develop a controller of a four high power, brushless direct current motors. The device was designed to receive and execute commands sent from master unit of the mobile robot. The robot is designed to work in a difficult external conditions, so emphasis has been put on reliability. The controller has been properly secured from exposure to undesirable factors (such as water, dust, EMI - electromagnetic interference) and also has three independent methods to control BLDC motors.

    Features:

  • drive up to 4 BLDC 0.5kW motors

  • 3 types of motor control - using Hall sensors, optical encoder or back EMF sensing

  • 36/48V input voltage

  • enclosure providing safe work in harsh conditions (dust, water, EMI, damage) and good heat dissipation

  • CAN communication over fibre connection (not tested)

Real-time measurement system with network interface utilizing asymmetric, 2-core processors of LPC4300 series

The goal of the thesis was to design and develop the universal measurement system with high precision timer. To achieve that the asymmetric, two core microcontroller LPC4357 has been used. It includes Cortex-M4 and M0 cores doing tasks, which has been properly distributed. The system is running FreeRTOS to keep all the time constraints of the specific tasks. The device is communicating through the industrial Ethernet. With this interface it receives precise time (IEEE 1588 - PTP) and power (IEEE 802.3af - PoE). The project was also designed to be used with a fibre Ethernet (100BASE-FX) with an external power source.

    Features:

  • Universal system with exchangeable measurement modules

  • High accuracy time synchronization through Precision Time Protocol (IEEE1588) - 90ns

  • Low cost - parts, enclosure, PCB manufacturing (2 layers board) - 150 €

  • External power supply or Pover over Ethernet (IEEE 802.3af)

  • Ethernet through UTP cable or fibre connection

  • Easy to extend and develop separately

  • Modular enclosure with DIN rail connector (Phoenix Contact - HBUS)

  • Asymmetric, 2-core microcontroller - measurement independent from main core load

  • Real-time system (FreeRTOS) on Cortex-M4 core

  • External flash memory on the measurement board containing Cortex-M0 core application (SPIFI)

Two-wheeled self-balancing robot

The goal of this group project was to build a self-balancing robot using a LEGO Mindstorms set. The mathematical model was created from an inverted pendulum using Lagrangian mechanics. A LQ-controller was designed and simulated in Simulink and then implemented to the real system using two methods. First by auto-generated code, second - with regular C programming.

I was responsible for C code implementation and wireless communication. RF access point didn't have I2C available, so there was a need of another translating board.
